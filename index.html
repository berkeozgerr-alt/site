Burası, klasik web sitemizin ana içerik alanıdır. HTML, sayfayı yapılandırmak için kullanılırken, CSS bu metnin nasıl görüneceğini belirler. Bu basit yapı, tüm modern web sitelerinin temelini oluşturur. Amacımız, en temel adımları atarak bir web sayfası oluşturmaktır.

Navigasyon menüsünü (Ana Sayfa, Hakkımızda vb.) kullanarak farklı sayfalara bağlantı verebilirsiniz. Şimdilik hepsi aynı sayfaya dönüyor.
